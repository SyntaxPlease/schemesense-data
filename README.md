# Government Data Intelligence Layer

Prepared by: Muhammad Ameen

## Overview
This repository contains the structured Maharashtra government scheme dataset used in the SchemeSense AI-powered welfare advisor.

## Dataset Details
- 20 Government Schemes
- Structured JSON format
- Normalized eligibility rules
- Clean metadata fields
- Verified official links

## Purpose
This dataset powers:
- Conversational scheme discovery
- Visual eligibility breakdown (✓ / ✗)
- AI rejection reasoning engine
- Backend eligibility processing

## Structure Example

Each scheme contains:
- scheme_id
- scheme_name
- category
- short_description
- benefit_amount
- eligibility_rules
- documents_required
- application_link
- department
- state
- last_verified
